# Prisma Social Manager

Prisma Social Manager is a tool designed to work with Facebook and Instagram APIs for analyzing and managing social media content.

## Features

- Access Facebook Page data (posts, videos, reels)
- Access Instagram Business data (posts, reels, insights)
- Analyze engagement metrics
- Automate content workflows

## Technologies

- Meta Graph API (Facebook & Instagram)
- Backend: Python
- API-first architecture

## Data & Privacy

This project only accesses data that is explicitly permitted via the Meta APIs.

No personal user data is sold or shared with third parties.

For more details, see the Privacy Policy:  
https://pyvoyager.github.io/Privacy-Policy/
